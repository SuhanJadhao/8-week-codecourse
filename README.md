## I. THE OBJECTIVE
To execute an intensive 8-week engineering sprint. This repository tracks the evolution from `HTML/CSS Beginner` to `Full-Stack AI Engineer`. We are building **AetherOS**, a distributed intelligence SaaS, utilizing the modern AI stack.

## II. ARCHITECTURAL STACK

### Frontend & UI
* **Next.js 15:** React Server Components & App Router
* **Tailwind CSS:** Utility-First Design
* **Shadcn UI:** Accessible Primitives

### Backend & AI
* **Supabase:** PostgreSQL + Auth + Row Level Security (RLS) + Vector Search
* **Vercel AI SDK:** LLM Orchestration & Streaming
* **Claude 3.5 Sonnet:** Core Intelligence Engine

## III. THE 8-WEEK PROTOCOL

* **Phase 1: Foundation (Weeks 1-2)** - Next.js App Router, React Hooks, and Generative UI via `v0.dev`.
* **Phase 2: Data & Intelligence (Weeks 3-4)** - Supabase Auth, RLS, and Vercel AI SDK streaming.
* **Phase 3: RAG & Agents (Weeks 5-6)** - Vector Similarity search (`pgvector`) and Multi-step tool execution.
* **Phase 4: Deployment & Evals (Weeks 7-8)** - Production hardening, cost optimization, and CI/CD.

## IV. ENGINEERING STANDARDS
1. **Zero Tolerance for `any`:** Strict TypeScript interfaces only.
2. **Agentic Workflow:** Windsurf and Claude are used for acceleration, not as a replacement for architectural understanding.
3. **Atomic Commits:** Every daily milestone must correspond to a clean, descriptive push.
4. **The BITS Standard:** We do not watch tutorials passively. We read documentation, break things, and rebuild them stronger.
